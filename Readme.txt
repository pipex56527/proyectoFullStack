Para instalar paquetes por medio de package.json ejecutar el siguiente comando:
>>npm i

Para visualizar lo anterior en documentacion, visitar 
https://www.digitalocean.com/community/tutorials/how-to-use-node-js-modules-with-npm-and-package-json-es
https://victorroblesweb.es/2018/01/02/instalar-dependencias-con-npm-api-restful-nodejs/

No olvidar instalar nodemon con el comando (aunque este ya fue agregado a package.json, deberia instalarse solo)
>> npm install nodemon --save-dev
